autoclicker.py
A GUI auto-clicker that starts/stops clicking with a button or the F6 hotkey. When active, it continuously clicks with a ~1 ms delay (plus a small random variation).

test.py
A clicks-per-second (CPS) test application. The user selects a test duration, then taps the main button as fast as possible during the test. At the end, it displays total clicks and average CPS.
